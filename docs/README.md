## 기능 목록 구현

- [ ] 돈을 입력하면 해당 돈 만큼의 로또 출력하기

  - [ ] 돈이 1000원으로 나누어지는가?
  - [ ] 구입한 돈 만큼의 로또 개수 구하기
  - [ ] 개수만큼 6자리 랜덤 로또 만들기

- [ ] 당첨 번호 6자리 입력(Lotto)

  - [x] 각 번호가 1 ~ 45인가?
  - [x] 번호는 6개인가?
  - [x] 중복이 존재하는가?

- [ ] 보너스 번호 1자리 입력

  - [ ] 각 번호는 1 ~ 45인가?

- [ ] 당첨 번호 + 보너스 번호와 내가 산 로또를 비교, 결과값 출력

  - [ ] 당첨 번호 + 보너스 번호와 내가 산 로또를 비교 (1 ~ 5등인지 결과값 나타내기)
  - [ ] 1 ~ 5등 결과값을 정해진 형식에 맞춰서 출력
